# Anagrams

- anagram의 의미 : 같은 글자(character)로 구성된 서로 다른 문자열(string) 말한다.

  > ex) cinema & iceman

- **문제)** 입력받은 문자열 2개가 `anagram 인지 아닌지`를 판별하시오. anagram 일 경우 true, 그렇지 않으면 false를 리턴하시오.
