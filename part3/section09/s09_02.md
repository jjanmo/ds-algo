# 그래프 경로 탐색

> 경로 탐색에서 사용하는 2가지 접근 방법에 대해서

노드 사이의 관계가 배열로 주어지는 경우, 이를 2차원 행렬의 `방향 그래프`로 변환한다. (시간복잡도에 따라서) 인접행렬 혹은 인접리스트로 변환하여 문제를 풀게된다.

주어진 n의 개수가 많아지면, 인접리스트로 문제를 푸는게 효율성이 더 높다. 왜냐하면 노드간에 연결된 곳만 순회하여 확인하기 때문이다.
